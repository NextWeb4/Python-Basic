## 使用示例

仓库没有声明 Python 版本、依赖清单、虚拟环境工具或统一命令。请先审查脚本，再使用兼容的解释器逐个运行编号示例。

这是历史练习集，部分示例使用当前 Python 已移除的 API，例如 `time.clock()`。实例 095 导入第三方 `dateutil`，绘图示例使用 GUI 模块并可能需要桌面会话。实例 097 至 099 会写入本地文件；运行前应检查源码，并为文件写入练习使用可丢弃的工作目录。

`.docx` 文件是参考文档，不是运行时依赖，也不需要构建。
