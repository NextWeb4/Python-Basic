## アーカイブの範囲

本リポジトリは初期の学習順序を保存したもので、インストール可能な Python パッケージではありません。各番号付きスクリプトは、前提、入力、出力、副作用がそれぞれ異なる独立した教材です。`.docx` は参照資料であり、ランタイムモジュールではありません。

100 ケースすべてに共通する対応 Python バージョン、依存関係マニフェスト、仮想環境ツール、パッケージマネージャー、共通エントリーポイント、互換性保証はありません。

## 監査済みインベントリ

更新したデフォルトブランチのツリーには、**1 ディレクトリ、113 ファイル**があります。

| 形式 | 数 | 用途 |
| --- | ---: | --- |
| `.py` | 100 | `Python100Cases-master/001.py` から `100.py` |
| `.docx` | 9 | 構文、ライブラリ、演習、ELIZA、クローラー題材、ケース参照文書 |
| `.md` | 4 | ルートの多言語 README とリポジトリ規則 |

ツリーに画像、依存関係ロックファイル、テストディレクトリ、Notebook、パッケージ設定、CI ワークフローはありません。

## 当時の利用環境

選んだスクリプトを最後まで読んでから、対応するインタープリターを使ってください。コレクションが一つの Python 2 または Python 3 リリースを対象としているかは確認できていません。

確認済みの互換性例外には、削除された `time.clock()` の使用、`095.py` の外部 `dateutil` import、デスクトップセッションを必要とする GUI・描画例、`097.py` から `099.py` のファイル書き込みがあります。他のケースは未実行であり、追加要件が存在する可能性があります。

## ケースごとの安全な手順

1. 一つの番号付きファイルを選び、import、入力処理、出力、ループ、ファイル操作を読みます。
2. インタープリター構文、外部 import、GUI 要件、読み取り・上書き対象ファイルを確認します。
3. 合成入力と破棄可能な作業ディレクトリを使い、そのケースだけを実行してインタープリター版を記録します。
4. 演習の意図と出力を比較し、失敗や現代化の選択をそのケースだけに記録します。
5. 生成ファイルを削除し、一つの成功を残り 99 ケースの検証結果として扱いません。

リポジトリ共通の実行コマンドはありません。確認済みの一例には直接インタープリターを使える場合がありますが、全体で互換性のある方法として記載しないでください。

## 互換性と副作用

- `time.clock()` など削除済み API は、現在の Python で失敗する場合があります。
- `095.py` は `dateutil` を必要としますが、依存関係マニフェストも承認済みバージョンもありません。
- GUI・描画例は、画面のないセッションで失敗したり、プラットフォームごとに挙動が変わったりします。
- `097.py` から `099.py` は、現在の作業ディレクトリにファイルを作成・上書きする可能性があります。
- 歴史的な教材には、欠陥、古い書き方、文字コードの前提、本番利用に適さない挙動が含まれる場合があります。
- バイナリ `.docx` にはリンクや埋め込み内容が含まれる可能性があり、通常の文書安全対策が必要です。

## 検証の範囲

リポジトリ共通のテスト、ビルド、lint、format コマンドは見つかっていません。今回の文書監査では 100 スクリプトを実行していないため、完全互換の Python バージョンは表明しません。

ケースを変更する場合は、入力、期待・観測出力、インタープリター版、依存関係版、生成ファイル、後片付けを記録してください。そのケース専用の確認を追加しても、アーカイブ全体のテストスイートとは説明しません。

## 保守とコントリビューション

変更は一つのケース、または明確な証拠のある文書修正に限定します。番号付きファイル名と各例の独立性を維持し、一つの削除 API を直すために 100 スクリプトを一括変更したり、依存関係を暗黙に追加したりしないでください。

English、简体中文、日本語の README は、章順、数値、パス、リンク、互換性の事実、副作用の警告を一致させてください。
