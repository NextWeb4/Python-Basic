## サンプルの利用方法

Python バージョン、依存関係マニフェスト、仮想環境ツール、共通コマンドは宣言されていません。スクリプトを確認し、対応するインタープリターで番号ごとに実行してください。

これは歴史的な演習集です。`time.clock()` など、現在の Python では削除された API を使う例があります。ケース 095 は外部パッケージ `dateutil` を import し、描画例は GUI モジュールとデスクトップ環境を必要とする場合があります。ケース 097 から 099 はローカルファイルを書き込むため、実行前にソースを確認し、破棄可能な作業ディレクトリを使用してください。

`.docx` は参照文書であり、実行時依存関係ではありません。ビルドも不要です。
