## What is included

- `Python100Cases-master/001.py` through `Python100Cases-master/100.py`.
- Introductory documents covering syntax, common libraries, exercises, ELIZA, and a NetEase Cloud Music crawler topic.
- The original README's detailed Chinese problem statements and sample implementations, now represented by the corresponding numbered script files.

## Working with the examples

No Python version, dependency manifest, virtual-environment tool, or repository-wide command is declared. Review and run one numbered script at a time with a compatible interpreter.

The collection is historical. Some examples use APIs removed from current Python versions, including `time.clock()`. Case 095 imports the third-party `dateutil` package, while drawing examples use GUI modules and may require a desktop session. Cases 097 through 099 write local files. Inspect a script before running it and use a disposable working directory for file-writing exercises.

The `.docx` files are reference documents rather than runtime dependencies. No build step is required for them.
